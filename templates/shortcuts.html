Step 1: Configure Server URL

When you install the shortcut, you'll be asked for your server URL:

http://192.168.1.87:5001

This is your local Flask server. Make sure it's running before using shortcuts.

Step 2: Test the Shortcut

After installation:

  1. Say "Hey Siri, code this"
  2. Wait for Siri to start listening
  3. Say: "Build me a function that validates email addresses"
  4. Wait 10 seconds
  5. Siri will say "Code is ready in your clipboard"
  6. Paste anywhere (Cmd+V)

  • "Code this" ‚Üí "Build me X"
