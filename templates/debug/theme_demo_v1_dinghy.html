ðŸ“– Usage Instructions

1. Include CSS and JS in your template

<link rel="stylesheet" href="/static/css/theme.css"> <script src="/static/js/theme-toggle.js"></script>

2. Inject brand colors

<style> :root { --brand-primary: {{ '{{ theme_primary }}' }}; --brand-secondary: {{ '{{ theme_secondary }}' }}; --brand-accent: {{ '{{ theme_accent }}' }}; } </style>

3. Use CSS variables in your styles

.my-card { background: var(--bg-card); color: var(--text-primary); border: 1px solid var(--border-light); } .my-button { background: var(--gradient-primary); color: var(--text-inverse); }

4. Available CSS Variables

/* Colors */ --color-primary, --color-secondary, --color-accent /* Backgrounds */ --bg-page, --bg-surface, --bg-elevated, --bg-card /* Text */ --text-primary, --text-secondary, --text-tertiary, --text-inverse /* Borders */ --border-light, --border-medium, --border-dark /* Status */ --success, --warning, --error, --info /* Shadows */ --shadow-sm, --shadow-md, --shadow-lg, --shadow-xl /* Gradients */ --gradient-primary, --gradient-surface

âš¡ JavaScript API

Control theme programmatically via window.themeSystem

// Toggle theme window.themeSystem.toggle(); // Set specific theme window.themeSystem.setTheme('dark'); window.themeSystem.setTheme('light'); // Get current theme const theme = window.themeSystem.getTheme(); // 'light' or 'dark' // Check if dark mode if (window.themeSystem.isDark()) { console.log('Dark mode is active'); } // Listen for theme changes window.addEventListener('themechange', (e) => { console.log('Theme changed to:', e.detail.theme); });
