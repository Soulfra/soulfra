## What This Does
Say "Hey Siri, create content" ‚Üí Talk for 5 minutes ‚Üí AI creates blog/tweets/video/docs

## Installation Steps

### 1. Open iOS Shortcuts App

### 2. Create New Shortcut

### 3. Add These Actions (in order):

ACTION 1: Record Audio
- Search for "Record Audio"
- Set "Stop Recording After" to 300 seconds (5 minutes)
- (This gives you time to ramble about any topic)

ACTION 2: Get Contents of URL
- URL: http://192.168.1.87:5001/api/voice/content-gen
- Method: POST
- Headers: Content-Type: application/json
- Request Body: JSON
    {
      "transcription": "[Audio from previous action]",
      "format": "all"
    }

ACTION 3: Get Dictionary Value
- Get value for key: content
- From: Contents of URL

ACTION 4: Get Dictionary Value (nested)
- Get value for key: blog
- From: Dictionary Value (from previous action)

ACTION 5: Copy to Clipboard
- Copy: Dictionary Value (blog post)

ACTION 6: Create Note
- Search for "Create Note"
- Note: [Contents of URL] (the full response)
- Title: "Voice Content - [Current Date]"
- (This saves ALL formats: blog, tweets, video, docs)

ACTION 7: Show Notification
- Title: "Content Ready"
- Body: "Blog post in clipboard, all formats in Notes"

### 4. Name Your Shortcut
Rename to: "Create Content"

### 5. Add to Siri
- Record phrase: "Create content"

## Usage

1. Say: "Hey Siri, create content"
2. Siri starts recording (up to 5 minutes)
3. Talk about anything:
   - Frustrations
   - Ideas
   - Observations
   - Technical concepts
   - Stories
4. Let Siri finish recording
5. Wait 20-30 seconds for AI processing
6. Notification appears
7. Blog post is in clipboard
8. All formats saved in Notes app

## Formats Breakdown

### Blog Post
- Catchy title
- Introduction (hook)
- 3-5 main points
- Conclusion
- Markdown formatted

### Tweet Thread
- 5-7 tweets
- Numbered (1/, 2/, 3/)
- Max 280 chars each
- Engaging hooks

### Video Script
- [INTRO - 0:00-0:15] Hook
- [MAIN - 0:15-2:00] Key points
- [OUTRO - 2:00-2:30] CTA

### Technical Docs
- Title
- Overview
- Usage
- Examples
- Markdown formatted

## Advanced: Single Format

If you only want one format (e.g., just tweets):

In "Get Contents of URL" action, change:
  "format": "all"
To:
  "format": "tweet"

Options:
- "blog" - Just blog post
- "tweet" - Just tweet thread
- "video" - Just video script
- "docs" - Just documentation

## Troubleshooting

Q: Recording stops early
A: Check recording time is set to 300 seconds (5 min)

Q: Content is too short
A: Talk more! AI needs at least 2 minutes of content

Q: Blog post doesn't match my voice
A: Add more personality to your rambling. AI mimics your tone.
