How It Works (Technical)

Simple Flow

  1. User scans QR code ‚Üí Opens branded widget
  2. Chats with AI ‚Üí Conversation stored
  3. Types /generate post ‚Üí AI creates blog post from chat
  4. Neural network classifies ‚Üí Assigns to correct brand
  5. Auto-sent to newsletter ‚Üí Subscribers get email

Stack

  • Backend: Python + Flask + SQLite
  • AI: Ollama (local, no API costs)
  • ML: Custom neural networks (scikit-learn)
  • Frontend: Vanilla JavaScript (no frameworks)

Zero cloud dependencies. Runs on a $5/month VPS.
