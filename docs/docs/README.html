AI-powered platform where development happens in public. Python-only, SQLite-based.

This project has TWO different servers you can run:

Port 5001 - Main Flask App (app.py)

python3 app.py
# Visit: http://localhost:5001
  • Full-featured content platform
  • Blog posts, comments, user profiles
  • AI reasoning from 4 personas (CalRiven, DeathToData, TheAuditor, Soulfra)
  • Flask + Jinja2 templates
  • Uses: soulfra.db

Port 8888 - Zero Dependencies Demo (soulfra_zero.py)

python3 soulfra_zero.py
# Visit: http://localhost:8888
  • Pure stdlib HTTP server (no Flask)
  • Neural network training dashboard
  • Multi-format output (JSON/CSV/TXT/HTML/RTF/Binary)
  • Zero external dependencies - demonstrates what's possible with pure Python
  • Uses: soulfra.db (read-only)

Both servers access the same SQLite database (soulfra.db) but serve different purposes. Choose port 5001 for normal development, port 8888 for zero-dependency demos.

What This Is

A self-documenting development platform: - Posts & comments with AI reasoning - Soul system (digital identity based on activity) - Machine learning (Python stdlib only, no external libs) - Database-first architecture (everything in SQLite) - Build-in-public automation - No Node.js, no PostgreSQL, no external ML libraries
