## Troubleshooting

### "Git push failed"

Check if git remote is configured:
```bash
cd output/soulfra
git remote -v
# Should show: origin  https://github.com/Soulfra/soulfra.git
```

If missing:
```bash
git remote add origin https://github.com/Soulfra/soulfra.git
```

### "Site not updating on soulfra.com"

1. Check GitHub Pages is enabled:
   - Go to repo â†’ Settings â†’ Pages
   - Source should be "main" branch

2. Check CNAME:
   - Should contain: `soulfra.com`

3. Check DNS (on your domain registrar):
   - Add CNAME record: `soulfra.com` â†’ `soulfra.github.io`
   - Or A record: `@` â†’ GitHub Pages IPs

### "Static export failed"

Check export_static.py works manually:
```bash
python3 export_static.py --brand soulfra
```

If errors, check database has posts:
```bash
sqlite3 soulfra.db "SELECT COUNT(*) FROM posts WHERE brand_id=1"
```
