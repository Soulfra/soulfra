üìã How It Works

HTML5 <audio> tag - Works on all modern browsers

  • controls - Shows play/pause, volume, scrubber
  • preload="metadata" - Loads duration without downloading whole file
  • <source> - Specify audio file + format
  • Multiple sources = fallback if browser doesn't support one format

üìù Code Examples

WAV File:

<audio controls>
    <source src="recording.wav" type="audio/wav">
    Your browser doesn't support audio.
</audio>

WebM File:

<audio controls>
    <source src="recording.webm" type="audio/webm">
    Your browser doesn't support audio.
</audio>

MP3 File:

<audio controls>
    <source src="recording.mp3" type="audio/mpeg">
    Your browser doesn't support audio.
</audio>

With Multiple Formats (fallback):

<audio controls preload="metadata">
    <source src="recording.webm" type="audio/webm">
    <source src="recording.mp3" type="audio/mpeg">
    <source src="recording.wav" type="audio/wav">
    Your browser doesn't support audio.
</audio>
