# GitHub Profile README Setup Instructions

## What is a GitHub Profile README?

A special repository that displays on your GitHub profile page when you visit `github.com/soulfra`.

## Step 1: Create the Special Repository

1. Go to: https://github.com/new
2. Repository name: **soulfra** (must match your GitHub username exactly)
3. Description: "My GitHub profile README"
4. Set to **Public** (required for profile README to work)
5. Check "Add a README file"
6. Click "Create repository"

## Step 2: Upload Your README

After creating the repo:

1. Go to your new repo: https://github.com/soulfra/soulfra
2. Click on `README.md`
3. Click the pencil icon (Edit this file)
4. Copy the contents of `README.md` from this directory
5. Paste into the GitHub editor
6. Scroll down and click "Commit changes"

## Step 3: Verify It Works

1. Go to your profile: https://github.com/soulfra
2. You should see your README displayed at the top of your profile!

## Step 4: Customize

Edit the README to add:
- Your actual GitHub username (replace 'soulfra')
- Real links to your domains
- Your email address
- Any custom badges or widgets

## Optional: Add Gacha Simulator Widget

If you want the interactive gacha simulator:

1. Create `gacha.html` in the repo
2. Copy the gacha simulator code (see GACHA_WIDGET.html in this directory)
3. Link to it from your README

## Troubleshooting

**README not showing on profile?**
- Make sure repo name **exactly** matches your username
- Make sure repo is **public**
- Make sure file is named **README.md** (case-sensitive)
- GitHub can take a few minutes to update

**Want to test locally first?**
- Preview your README.md using VS Code or any Markdown previewer
- Check markdown syntax: https://github.com/adam-p/markdown-here/wiki/Markdown-Cheatsheet
