üí° How to Use This

This is a static debug dashboard that works on the live site (soulfra.com) without requiring a Flask server.

To generate reports locally:

python3 debug_system.py --routes - System debug
python3 brand_mapper.py - Brand analysis
python3 ccna_study.py - CCNA graphs
python3 build-content.py - Blog posts

To deploy to live site:

./deploy-tools.sh - Deploy all reports to soulfra.com

Mobile debugging: This page works on iPhone! Bookmark it for on-the-go debugging.
