About Soulfra

Soulfra is a newsletter exploring AI, privacy, and the future of technology. We believe in:

  • Open source - Everything we build is public
  • Privacy first - Your data belongs to you
  • No tracking - No analytics, no cookies, no BS
  • Simple tech - Python, SQLite, Markdown. That's it.

Why This Newsletter Exists

Most platforms lock you in. Substack takes 10% forever. Medium owns your audience. Twitter can ban you anytime.

We built Soulfra Simple to prove you can own your content, own your subscribers, and deploy everything for $0/month using GitHub Pages.

How It Works

  • Posts are written in Markdown
  • Static site generated with Python
  • Hosted on GitHub Pages (free)
  • Emails sent via SMTP (Gmail)
  • Database is just SQLite
